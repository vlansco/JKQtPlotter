36 /*! \brief This implements a single vertical <a href="https://en.wikipedia.org/wiki/Violin_plot">Violin Plot</a> as a "geometric element"
37  \ingroup jkqtplotter_statgraphs
38  \ingroup jkqtplotter_diverse
39 
40  Draws a vertical violin plot in one of the following styles:
41 
42  \image html JKQTPGraphViolinplot_SmoothViolin.png
43 
44  \image html JKQTPGraphViolinplot_StepViolin.png
45 
46  \image html JKQTPGraphViolinplot_BoxViolin.png
47 
48 
49  This graph/plot element represents a single such plot. Its center is positioned at an x-coordinate provided by setPos().
50  The width of the full plot (left+right!) is given by JKQTPGraphViolinplotStyleMixin::setViolinWidthAbsolute().
51 
52  The data is represented partly by single values (e.g. setMedian(), setMin(), ...) and partly by the two columns set with
53  setViolinPositionColumn() and setViolinFrequencyColumn(). The latter two represent a density estimate (or histogram) of the
54  distribution represented by this plot, which is evaluated at the y-positions from setViolinPositionColumn(). For each such
55  y-position a (not necessarily normalized) frequency is given by the values in setViolinFrequencyColumn().
56 
57 
58  \image html JKQTPViolinplotHorizontalElement_elements.png
59 
60  \see \ref JKQTPlotterViolinplotGraphs
61 
62  */
