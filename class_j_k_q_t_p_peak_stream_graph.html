Detailed Description

interprets data as a stream of x- or y-positions (depending in yPeaks ) ans plots a line on the height baseline and upright lines from baseline to baseline+peakHeight at each position . This can be used to display a stream of photons with given arrivaltimes .

yPeaks=true
yPeaks=false
See also
Tutorial (JKQTPDatastore): Advanced 1-Dimensional Statistics with JKQTPDatastore
