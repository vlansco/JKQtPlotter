Detailed Description

This implements a single horizontal Violin Plot as a "geometric element".

Draws a horizontal violin plot in one of the following styles:

This graph/plot element represents a single such plot. Its center is positioned at an y-coordinate provided by setPos(). The width of the full plot (top+bottom!) is given by JKQTPGraphViolinplotStyleMixin::setViolinWidthAbsolute().

The data is represented partly by single values (e.g. setMedian(), setMin(), ...) and partly by the two columns set with setViolinPositionColumn() and setViolinFrequencyColumn(). The latter two represent a density estimate (or histogram) of the distribution represented by this plot, which is evaluated at the x-positions from setViolinPositionColumn(). For each such x-position a (not necessarily normalized) frequency is given by the values in setViolinFrequencyColumn().
