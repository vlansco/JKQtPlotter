Detailed Description

This JKQTPPlotObject is used to draw a line, optionally line-end decorations (aka arrows) are pssible, but switched off by default.

Note
This class support JKQTPPlotObject::DrawMode::DrawAsMathematicalCurve. If set, and non-linear axes are chosen, the line is drawn as a curves, instead of straight a straight line. In the mode JKQTPPlotObject::DrawMode::DrawAsGraphicElement the line is drawn as a straight line.

You can also activate line-end decorators (aka arrows) for this poly-line, by using code like this:

line->setHeadDecoratorStyle(JKQTPFilledDoubleArrow);
line->setTailDecoratorStyle(JKQTPCircleDecorator);

This results in arrows drawn at the start (aka x1/y1, =tail) and end (aka x2/y2, =head) of the line.

Note
The convenience class JKQTPGeoArrow activates line-end decorations (aka arows) by default and allows to select them in the constructor.
See also
JKQTPGeoArrow, Example (JKQTPlotter): Plotting Geometric Objects and Example (JKQTPlotter): Plotting Arrows
