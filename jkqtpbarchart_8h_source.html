35 /*! \brief This implements a bar graph with bars starting at \f$ yoverride \f$ to \f$ y=f(x) \f$
36  \ingroup jkqtplotter_barssticks
37 
38  This class plots a bargraph. This image explains the parameters:
39 
40  \image html bargraph_basics.png
41 
42  By default the sift parameter is, so the bar is centered at the x-value. The width is 0.9,
43  so adjacent bars are plotted with a small distance between them. It is possible to use these two parameters
44  to plot multiple bars for every x-value, by having on JKQTPSpecialLineHorizontalGraph object per
45  set of bars that belong together. For example for three bars per x-value one would set:
46  \verbatim
47  widthoverride.3
48  shift=-0.3 / 0 / +0.3
49  \endverbatim
50  This results in a bargraph, as shown here:
51 
52  \image html plot_bargraphverplot.png
53 
54 
55  You can use JKQTPlotter::addHorizontalBargraph() to add a series of bargraphs, where the width and shift are determined
56  automatically. The y-columns are given as a QVector<int> to this function.
57 
58  \see JKQTPBarHorizontalGraph, \ref JKQTPlotterBarcharts, jkqtpstatAddHHistogram1D(), jkqtpstatAddHHistogram1DAutoranged()
59  */
