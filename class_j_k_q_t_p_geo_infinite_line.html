Detailed Description

This JKQTPPlotObject is used to draw an infinite line.

an infinite line has a starting point and then goes on in a given direction until the end of the plotting range.

Note
This class support JKQTPPlotObject::DrawMode::DrawAsMathematicalCurve. If set, and non-linear axes are chosen, the line is drawn as a curves, instead of straight a straight line. In the mode JKQTPPlotObject::DrawMode::DrawAsGraphicElement the line is drawn as a straight line.

You can add a decorator to the head of the line (i.e. the given start point (x,y) ) iff this line is one-sided, i.e. two_sided ==false .

See also
Example (JKQTPlotter): Plotting Geometric Objects and Example (JKQTPlotter): Plotting Arrows
