51 /*! \brief this class parses a LaTeX string and can then draw the contained text/equation onto a <a href="http://doc.qt.io/qt-5/qpainter.html">QPainter</a>
52  \ingroup jkqtmathtext
53 
54 
55  JKQTMathText is a self-contained LaTeX-renderer for Qt. It is used to renderer
56  labels in JKQTPlotter/JKQTBasePlotter, but can be used independently.
57  The class does not depend on any library, except Qt.
58  In particular it actually parses a LaTeX string and draws it in pure C++. It does NOT rely
59  on an installed LaTeX for the rendering!
60 
61  \section JKQTMathTextUsage Usage
62  \subsection JKQTMathTextUsageDirect Direct Usage
63  This small piece of C++ code may serve as an example of the usage and capabilities of the class:
64  \code
65  // create a JKQTMathText object.
66  JKQTMathText mathText;
67 
68  // configure its properties to influence the rendering (e.g. fonts to use, font size, ...)
69  mathText.useXITS();
70  mathText.setFontSize(20);
71 
72  // parse some LaTeX code (the Schroedinger's equation)
73  mathText.parse("$\\left[-\\frac{\\hbar^2}{2m}\\frac{\\partial^2}{\\partial x^2}+V(x)\\right]\\Psi(x)=\\mathrm{i}\\hbar\\frac{\\partial}{\\partial t}\\Psi(x)$");
74 
75  // use the draw() methods to draw the equation using a QPainter (here onto a QPixmap)
76  QPainter painter;
77  QPixmap pix(600,400);
78  painter.begin(&pix);
79  mathText.draw(painter, Qt::AlignCenter, QRectF(0,0,pix.width(), pix.height()), false);
80  painter.end();
81  \endcode

112  \section JKQTMathTextSuppoertedLaTeX Supported LaTeX Subset
113  \subsection JKQTMathTextSuppoertedLaTeXSimple Simple Instructions
114  The supported LaTeX subset is listes below. Please note that some commands are used differently than in actual LaTeX.
115  For example \c \\bf is just a renamed form of \c \\textbf and used as \c \\bf{...} and NOT as \c {\\bf...} .
116 
117  - \c $ : enter/leave math mode
118  - \c \\bf{...} \c \\textbf{...} \c \\mathbf{...}: draw the contained text in bold face \image html jkqtmathtext_bold.png
119  - \c \\it{...} \c \\textit{...} \c \\mathit{...} : draw the contained text in italic face \image html jkqtmathtext_italic.png
120  - \c \\rm{...} \c \\textrm{...} \c \\mathrm{...} \c \\mbox{...} : draw the contained text in normal upright roman font face \image html jkqtmathtext_fonts.png
121  - \c \\sf{...} \c \\textsf{...} \c \\mathsf{...} : draw the contained text in normal upright sans-serif font face \image html jkqtmathtext_fonts.png
122  - \c \\script{...} \c \\textscript{...} \c \\mathscript{...} : draw the contained text in a script font face \image html jkqtmathtext_fonts.png
123  - \c \\sc{...} : draw the text in small caps \image html jkqtmathtext_sc.png
124  - \c \\ul{...} \c \\underline{...} \c \\underlined{...} : draw the text with underlining \image html jkqtmathtext_ul.png
125  - \c \\ol{...} \c \\overline{...} \c \\overlined{...} : draw the text with overlining \image html jkqtmathtext_ol.png
126  - \c \\tt{...} \c \\texttt{...} \c \\mathtt{...} : draw text in typewriter font \image html jkqtmathtext_fonts.png
127  - \c \\textcolor{color}{...} \c \\color{color} \c \\mathcolor{color}{...} : draw colored text \image html jkqtmathtext_colored.png
128  - \c \\boxed{...} : draw text with a box around it \image html jkqtmathtext_boxed.png
129  - \c \\colorbox{color}{...} : draw a colored box around text \image html jkqtmathtext_colorbox.png
130  - \c \\alpha ... : display the according greek letter \image html jkqtmathtext_greek.png
131  - \c ^{...} \c _{...} : display the contents of braces in superscript/subscript \image html jkqtmathtext_supersub.png
132  <br>Special subscript/superscript typesetting applies, when the sub/super follows \c \\sum \c \\Prod ...: \image html jkqtmathtext_specialsubsuper.png
133  - \c \\{ / \\} : display opening/closing brace
134  - \c \\_ : display underscore
135  - \c \\sum \c \\prod \c \\int ... : plot special symbol. Note that depending on the fontEncoding the available
136  symbols may differ (there are not all symbols defined in the MS Windows Symbol
137  font!). Best coverage should be given by Unicode font encoding with a good
138  unicode font installed!<br>\image html jkqtmathtext_symbols.png
139  - <code>\\vec{x} \\dot{x} \\ddot{x} \\overline{x} \\underline{x} \\hat{x} \\tilde{x} \\uul{x} \\ool{x} \\bar{x} \\arrow{x} </code>: Decorations over/under symbols \image html jkqtmathtext_mathdeco.png
140  - <code>\\verb{don't parse this _aaa\LaTeX} </code>: interpret enclosed text as verbose \image html jkqtmathtext_verb.png

143  \subsection JKQTMathTextSuppoertedLaTeXBraces Braces ...
144  Braces in math mode are adjusted in size, so they are a small bit (factor \c brace_factor ) larger than the contents.
145  To enable this you have to write braces with \c \\left and \c \\right. These types of braces are defined (slight
146  differences to LaTeX standard):
147  - \c \\left( \c \\right) : default meaning (), \image html jkqtmathtext_brace_round.png
148  - \c \\left[ \c \\right] : default meaning [], \image html jkqtmathtext_brace_rect.png
149  - \c \\left\\{ \c \\right\\} : default meaning {}, \image html jkqtmathtext_brace_curly.png
150  - \c \\left< \c \\right> : "averaging" braces , \image html jkqtmathtext_brace_tri.png
151  - \c \c \\left\\lfloor \c \\right\\rfloor : floor braces , \image html jkqtmathtext_brace_floor.png
152  - \c \\left~ \c \\right~ \c \\left\\lceil \c \\right\\rceil : ceil braces , \image html jkqtmathtext_brace_ceil.png
153  - \c \\left| \c \\right| : absolute value braces | |, \image html jkqtmathtext_brace_oneline.png
154  - \c \\left\| \c \\right\| : norm braces || ||, \image html jkqtmathtext_brace_dblline.png
155  - You can use \c \\left. or \c \\right. to have only right or only left brace
156  .

181  \section JKQTMathTextSuppoertedFonts Font Handling
182 
183  Several fonts are defined as properties to the class:
184  - A "roman" font used as the standard font ( setFontRoman() )
185  - A "sans-serif" font which may be activated with \c \\sf ... ( setFontSans() )
186  - A "typewriter" font which may be activated with \c \\tt ... ( setFontTypewriter() )
187  - A "script" font which may be activated with \c \\script ... ( setFontScript() )
188  - A greek font which is used to display greek letters \c \\alpha ... ( setSymbolfontGreek() )
189  - A symbol font used to display special (math) symbols. ( setSymbolfontSymbol() )
190  - A "roman" font used as the standard font in math mode ( setFontMathRoman() )
191  - A "sans-serif" used as sans serif font in math mode ( setFontMathSans() )
192  - A "blackboard" font used to display double stroked characters (setFontBlackboard() )
193  - A "caligraphic" font used to display caligraphic characters ( setFontCaligraphic() )
194  .
195 
196  These fonts are generic font classes, which font is actually used can be configured in JKQTMathText class with the \c set...() functions mentioned above. You can also use these functions to set the fonts used for math rendering in math-mode:
197  - useSTIX() use the STIX fonts from <a href="https://www.stixfonts.org/">https://www.stixfonts.org/</a> in math-mode<br>\image html jkqtmathparser_stix.png
198  - useXITS() use the XITS fonts from <a href="https://github.com/alif-type/xits">https://github.com/alif-type/xits</a> in math-mode. These are included by default in this library and also activated by default.<br>\image html jkqtmathparser_xits.png
199  - useASANA() use the ASANA fonts from <a href="https://ctan.org/tex-archive/fonts/Asana-Math/">https://ctan.org/tex-archive/fonts/Asana-Math/</a> in math-mode<br>\image html jkqtmathparser_asana.png
200  - useAnyUnicode() use generic Unicode fonts, e.g. "Arial" and "Times New Roman" in math-mode. You should use fonts that contain as many of the mathematical symbols as possible to ensure good rendering results.<br>using "Times New Roman": \image html jkqtmathparser_timesnewroman.png
201  <br>using "Arial": \image html jkqtmathparser_arial.png
202  <br>using "Courier New": \image html jkqtmathparser_couriernew.png
203  <br>using "Comic Sans MS": \image html jkqtmathparser_comicsans.png
204  .

206

208  - <b>math-mode (XITS fonts are used, whitespaces are mostly not drawn directly, symbol spacing is different)</b> \c $...$: <br>\image html jkqtmathparser_schreq_mathmode.png
209  - <b>normal mode (Times new Roman is used, whitespaces are evaluated directly)</b>: <br>\image html jkqtmathparser_schreq_normalmode.png
